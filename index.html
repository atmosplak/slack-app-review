Slack App Security Review Tool

Use this tool to determine the security review requirements for an incoming app request

Instructions:

  1. Navigate to Slack App Directory and enter app in search bar
  2. Click into the app to see more info
  3. Click the "Security & Compliance" tab to access the CSV file
  4. Download the CSV to your computer
  5. Upload the CSV file into the file uploader by clicking "Choose File" below
  6. Press the "Read CSV" button to determine security review requirements
